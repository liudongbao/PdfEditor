# PDF编辑器 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目背景
开发一款 Windows 桌面程序，支持把多个 PDF 合并成一个、删除单个PDF多余页面、调整页面顺序，维护文档目录，或者把 PDF 内容导出成 Word、PPT、Excel 等常用格式。

### 1.2 目标用户
主要面向日常需要处理 PDF 文件的办公用户，例如行政、人事、财务、教师、学生、项目管理人员，以及经常整理报告和资料的职场用户。

### 1.3 技术架构
- **开发框架**：.NET 9.0 + WPF
- **PDF 处理**：PoDoFo (C++ PDF 库，通过 C++/CLI 包装层调用)
- **Word 导出**：Xceed.Words.NET (DocX)
- **Excel 导出**：EPPlus 5.8.10
- **PPT 导出**：DocumentFormat.OpenXml 3.0.2
- **PDF 预览**：WebView2 + PDF.js
- **C++ 库管理**：vcpkg

## 2. 功能需求

### 2.1 已完成功能

#### 2.1.1 PDF 合并
- 支持选择多个 PDF 文件
- 支持调整合并顺序（上移/下移）
- 支持移除不需要的文件
- 进度条显示合并进度
- 合并完成后自动保存

#### 2.1.2 页面编辑
- 打开单个 PDF 文件
- 显示页面列表
- **页面预览**：左侧选择页面，右侧实时预览对应内容
- 删除不需要的页面
- 调整页面顺序（上移/下移）
- 保存编辑后的 PDF

#### 2.1.3 格式导出
- **Word 导出**：将 PDF 文本内容导出为 .docx 格式
- **Excel 导出**：将 PDF 文本内容按行导出为 .xlsx 格式
- **PPT 导出**：将 PDF 文本内容导出为 .pptx 格式

#### 2.1.4 目录维护（可视化编辑）
**界面布局**：
- 左侧：书签目录树形视图
- 右侧：PDF 预览区域 + 页面导航

**书签操作**：
- 添加书签（顶级/子书签）
- 编辑书签（标题、页码）
- 删除书签
- **快速添加**：滚动到当前页，一键添加书签
- **批量添加**：指定起止页码和标题前缀，批量创建书签

**书签样式**：
- 书签颜色设置（红、蓝、绿、黄、紫、默认）
- 书签加粗设置
- 应用样式到选中书签

**书签导航**：
- 上一页/下一页翻页
- 点击书签跳转到对应页面
- 书签搜索定位

**书签排序**：
- 拖拽书签调整顺序
- 自动按页码排序

**多级书签**：
- 支持创建子书签
- 树形层级显示
- 支持多级嵌套

**书签保存**：
- 将书签保存到 PDF 文件
- 支持保存为新文件

## 3. UI/UX 设计

### 3.1 整体布局
采用多标签页设计，分为四个主要功能区：

1. **PDF 合并** - 左侧文件列表 + 右侧操作按钮
2. **页面编辑** - 左侧页面列表 + 右侧预览区域
3. **格式导出** - 导出选项 + 导出按钮
4. **目录管理** - 左侧书签目录树 + 右侧 PDF 预览和导航

## 5. 发布说明

### 5.1 发布包类型

| 文件 | 大小 | 运行时需求 | 说明 |
|------|------|------------|------|
| `PdfEditor_v1.0.0.zip` | ~86 MB | ❌ 不需要 | 完整发布包，解压即用，无需安装任何依赖 |
| `publish/PdfEditor.exe` | - | ❌ 不需要 | 开发发布目录，需配合所有 DLL 文件使用 |

**所有发布包均为自包含版本（Self-Contained），已内置 .NET 9.0 运行时，无需用户额外安装。**

### 5.2 使用说明

1. 下载 `PdfEditor_v1.0.0.zip`
2. 解压到任意目录
3. 双击 `PdfEditor.exe` 运行
4. 如果被 Windows SmartScreen 阻止，点击"更多信息" → "仍要运行"

### 5.3 系统要求

- Windows 10/11 64 位系统
- 无需安装任何运行时（已内置 .NET 9.0）

### 5.4 已知问题

1. 发布包较大（约 86 MB），包含 .NET 9.0 运行时和 PoDoFo 原生依赖
2. 部分导出功能依赖第三方库，请确保遵守相关许可协议
3. PPT 导出功能使用 PDFium 渲染图片方式实现
