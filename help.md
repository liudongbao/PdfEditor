## 1. 软件简介

PDF 编辑工具是一款面向 Windows 办公用户的 PDF 桌面工具，提供以下核心功能：

- **PDF 合并**：将多个 PDF 文件合并成一个完整文档
- **页面编辑**：删除多余页面、调整页面顺序
- **目录维护**：可视化编辑 PDF 书签/目录
- **格式导出**：将 PDF 导出为 Word、Excel、PowerPoint 格式

## 5. PDF 合并

### 5.1 功能说明

将多个 PDF 文件按指定顺序合并成一个 PDF 文件。

### 5.2 操作步骤

1. 切换到「PDF 合并」标签页
2. 点击「添加文件」按钮，选择一个或多个 PDF 文件
3. （可选）调整文件顺序：
   - 选中文件，点击「上移」/「下移」按钮调整顺序
   - 支持按住 Ctrl 多选文件
4. （可选）移除不需要的文件：
   - 选中文件，点击「移除选中」按钮
5. 点击「开始合并」按钮
6. 选择保存位置和文件名
7. 等待合并完成

### 5.3 进度显示

- 进度条显示合并进度
- 状态栏显示当前处理状态
- 完成后弹出提示框

### 5.4 注意事项

- 合并顺序即最终文档的页面顺序
- 建议先按顺序添加，避免频繁调整
- 如果某个 PDF 文件损坏或加密，可能导致合并失败

## 6. 页面编辑

### 6.1 功能说明

打开单个 PDF 文件，查看页面列表，删除不需要的页面，调整页面顺序，然后保存。

### 6.2 操作步骤

1. 切换到「页面编辑」标签页
2. 通过以下方式之一打开 PDF：
   - 点击菜单栏「文件」→「打开 PDF」
   - 或直接在页面列表区域操作
3. 左侧显示页面列表，右侧显示预览
4. **删除页面**：
   - 在左侧列表中选中要删除的页面（支持 Ctrl 多选）
   - 点击「删除选中」按钮
5. **调整顺序**：
   - 选中页面，点击「上移」/「下移」按钮
6. **保存修改**：
   - 点击「保存修改」按钮
   - 选择保存位置（可另存为新文件）

### 6.3 页面预览

- 左侧选中某页后，右侧自动预览对应页面
- 预览使用 PDF.js 渲染，支持滚动查看
- 预览状态显示在预览区域下方

### 6.4 注意事项

- 删除页面后，建议先预览确认再保存
- 保存时默认另存为新文件，不会覆盖原文件
- 大文件预览可能需要几秒钟加载

## 7. 格式导出

### 7.1 功能说明

将 PDF 文件内容导出为其他格式，支持 Word、Excel、PowerPoint 三种格式。

### 7.2 支持的导出格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| Word | .docx | 提取 PDF 文本内容生成 Word 文档 |
| Excel | .xlsx | 将文本按行导入 Excel 表格 |
| PowerPoint | .pptx | 将每页 PDF 渲染为图片插入幻灯片 |

### 7.3 操作步骤

1. 切换到「格式导出」标签页
2. 点击「浏览...」按钮选择要导出的 PDF 文件
3. 在「导出格式」区域勾选需要的格式（可多选）
   - ☐ 导出为 Word (.docx)
   - ☐ 导出为 Excel (.xlsx)
   - ☐ 导出为 PowerPoint (.pptx)
4. 点击「开始导出」按钮
5. 选择保存位置
6. 等待导出完成

### 7.4 进度与状态

- 进度条显示导出进度
- 完成后弹出提示框，显示导出的文件数量
- 文件保存在所选目录下

### 7.5 导出效果说明

| 格式 | 导出效果 | 适用场景 |
|------|----------|----------|
| Word | 提取文本，保留段落顺序 | 需要编辑文本内容 |
| Excel | 每行文本对应一行单元格 | 数据表格类 PDF |
| PPT | 每页一张高清图片，排版完全一致 | 需要保留原格式展示 |

### 7.6 注意事项

- PPT 导出采用图片方式，生成的文件可能较大
- Word/Excel 导出为文本提取，复杂排版可能失真
- 建议导出后检查一下内容是否符合预期

## 8. 目录管理

### 8.1 功能说明

可视化编辑 PDF 书签（目录），支持添加、编辑、删除、排序、样式设置等操作。

### 8.2 界面布局

```
┌─────────────────┬─────────────────────────────────────┐
│   书签列表       │   快速添加书签   第 1/10 页  ◀▶    │
│  搜索框          ├─────────────────────────────────────┤
│                  │                                     │
│ ▶ 第1章          │                                     │
│   ▶ 1.1节        │        PDF 预览区域                  │
│     1.1.1        │                                     │
│     1.1.2        │                                     │
│   ▶ 1.2节        │                                     │
│ ▶ 第2章          │                                     │
│ ▶ 第3章          │                                     │
│                  │                                     │
│ [添加] [编辑]    │  书签样式: [红色▼] [ ]加粗 [应用样式] │
│ [删除] [更多]    │                                     │
└─────────────────┴─────────────────────────────────────┘
```

### 8.3 打开 PDF

在「目录管理」标签页中，可通过以下方式打开 PDF：
- 点击菜单栏「文件」→「打开 PDF」
- 打开后左侧自动显示 PDF 已有的书签
- 右侧显示 PDF 预览

### 8.4 添加书签

#### 方式一：快速添加当前页

1. 在右侧预览区域翻到目标页面
2. 在「快速添加书签」输入框中输入标题
3. 点击「添加」按钮
4. 书签将自动添加到左侧列表，指向当前页面

#### 方式二：手动添加

1. 点击左侧「添加」按钮
2. 在弹出的对话框中输入：
   - 书签标题
   - 目标页码
3. 点击「确定」

#### 方式三：批量添加

1. 点击「批量添加」按钮
2. 设置参数：
   - 起始页码
   - 结束页码
   - 标题前缀（如「第」）
   - 标题后缀（如「章」）
3. 点击「确定」，自动创建多个书签

### 8.5 添加子书签

1. 选中要添加子书签的父书签
2. 点击「▼」更多按钮，选择「添加子书签」
3. 输入标题和页码
4. 点击「确定」

### 8.6 编辑书签

1. 选中要编辑的书签
2. 点击「编辑」按钮
3. 修改标题或页码
4. 点击「确定」

### 8.7 删除书签

1. 选中要删除的书签（支持多选）
2. 点击「删除」按钮
3. 确认删除

> 注意：删除父书签时，其子书签也会一并删除。

### 8.8 调整书签顺序

#### 方式一：拖拽排序

1. 鼠标按住要移动的书签
2. 拖动到目标位置
3. 松开鼠标即可

#### 方式二：更多菜单

点击「▼」更多按钮，选择：
- 上移
- 下移
- 升级（提升一级）
- 降级（降低一级）

### 8.9 书签搜索

1. 在搜索框中输入关键词
2. 书签列表自动过滤匹配项
3. 清空搜索框恢复完整列表

### 8.10 书签样式

#### 设置颜色

1. 选中一个或多个书签
2. 在底部「书签样式」区域选择颜色：
   - 红色
   - 蓝色
   - 绿色
   - 黄色
   - 紫色
   - 灰色
3. 点击「应用样式」

#### 设置加粗

1. 选中一个或多个书签
2. 勾选「加粗」复选框
3. 点击「应用样式」

> 注意：书签样式是否显示取决于 PDF 阅读器的支持，部分阅读器可能不显示颜色和加粗样式。

### 8.11 页面导航

#### 翻页

- 点击「◀」按钮：上一页
- 点击「▶」按钮：下一页

#### 跳转到书签

1. 在左侧书签树中点击某个书签
2. 右侧预览自动跳转到对应页面

#### 跳转到指定页

1. 点击「跳转到」按钮
2. 输入页码
3. 点击「确定」

### 8.12 保存书签

1. 编辑完成后，点击菜单栏「文件」→「保存」或「另存为」
2. 选择保存位置
3. 等待保存完成
4. 保存后的书签可在其他 PDF 阅读器中正常显示和跳转

### 8.13 注意事项

- 保存书签会修改 PDF 文件，建议先备份原文件
- 建议使用「另存为」保存为新文件
- 书签跳转使用「适合页面」模式
- 支持多级书签，层级无限制
- 书签标题支持中文

## 9. 常见问题

### 9.1 运行相关

**Q: 双击 exe 没反应？**
A: 可能被 Windows 安全中心阻止了。右键点击 exe → 属性 → 勾选「解除锁定」 → 确定。

**Q: 首次启动很慢？**
A: 首次运行需要初始化 WebView2 组件，大约需要几秒钟，属于正常现象。

**Q: 提示缺少 .NET 运行时？**
A: 本软件为自包含版本，不需要额外安装 .NET 运行时。如果出现此提示，请确认下载的是完整发布包。

### 9.2 PDF 打开相关

**Q: 打开 PDF 报错「Unknown error occurred while loading PDF」？**
A: 可能的原因：
- PDF 文件已损坏
- PDF 文件被加密（有密码保护）
- PDF 文件正在被其他程序占用
- 文件名或路径包含特殊字符

**Q: PDF 预览不显示内容？**
A:
- 检查 WebView2 是否正常初始化
- 等待几秒钟让页面加载
- 确认 pdf.min.js 文件存在于程序目录

### 9.3 书签相关

**Q: 保存的书签在其他 PDF 阅读器中不显示？**
A:
- 确保使用「另存为」保存了修改
- 确认 PDF 文件不是只读属性
- 部分精简版 PDF 阅读器可能不支持书签

**Q: 书签跳转不准确？**
A: 目前书签跳转为「适合页面」模式，会跳转到整页。后续版本会支持精确位置跳转。

### 9.4 导出相关

**Q: Word 导出后排版乱了？**
A: Word 导出是文本提取模式，只能提取文字内容，复杂排版（图片、表格、分栏等）无法完全保留。

**Q: PPT 导出后文件很大？**
A: PPT 导出使用高清图片方式，每页约 100-300 KB，属于正常情况。

**Q: 导出进度条不动？**
A: 大文件导出可能需要较长时间，请耐心等待。如果超过 5 分钟没有响应，可以关闭重试。

### 9.5 其他

**Q: 软件会联网吗？**
A: 本软件完全离线运行，不会上传任何数据。

**Q: 支持加密 PDF 吗？**
A: 当前版本暂不支持加密（有密码）的 PDF 文件。
